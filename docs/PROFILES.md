# Пресеты

## Цель пресетов

Пресеты дают готовые наборы JVM-настроек под разные сценарии:

- `compat` — максимум совместимости;
- `balanced` — стабильный baseline для большинства ПК;
- `performance` — более агрессивный throughput;
- `ultra` — high-end сценарии.

## Практика выбора

1. Начните с `balanced`.
2. Сделайте минимум 2 реальных запуска на каждый кандидат.
3. Запустите `cli.exe --benchmark`.
4. Проверьте confidence в ранжировании.

## Где смотреть результаты

- `logs/presets/<preset>.jsonl`
- `logs/wrapper.log`

## Важно

- Короткие bootstrap-запуски не отражают реальный FPS.
- Без игровых метрик ranking остается рабочим, но менее точным.
