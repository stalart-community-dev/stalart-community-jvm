# Пресеты

## Цель пресетов

Пресеты дают готовые наборы JVM-настроек под разные сценарии:

- `compat` — максимум совместимости;
- `balanced` — стабильный baseline для большинства ПК;
- `performance` — более агрессивный throughput;
- `ultra` — high-end сценарии.

## Практика выбора

1. Начните с `balanced`.
2. Примените `Apply Recommended Config` (или `cli.exe --autotune`).
3. При необходимости вручную сравните соседние пресеты на одинаковой сцене.

## Где смотреть результаты

- `logs/wrapper.log`

## Важно

- Сначала используйте рекомендованный пресет по железу.
- Тонкая ручная настройка имеет смысл только после длительного реального теста в игре.
