# Подбор оптимального конфига

## Цель

Подобрать лучший пресет для конкретного ПК без встроенного сбора метрик в wrapper.

## Базовая стратегия

1. Применить `Apply Recommended Config` / `cli.exe --autotune`.
2. Использовать `balanced` как базовый пресет.
3. Для слабых систем пробовать `compat`, для сильных — `performance`/`ultra`.

## Ручная валидация

Проверяйте пресеты вручную:
- одинаковая карта/сцена;
- одинаковые графические настройки;
- минимум 10-15 минут на пресет;
- оценка по реальному фреймтайму/ощущению микрофризов.

## Рекомендации по чистоте измерений

- одинаковая сцена/маршрут теста;
- одинаковые настройки графики;
- закрытые фоновые тяжелые процессы;
- не сравнивать по сверхкоротким сессиям;
- после выбора пресета фиксируйте его через `Select Config`.
