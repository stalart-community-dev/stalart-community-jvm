# Устранение неполадок

## Быстрая диагностика

1. Проверьте `cli.exe --status`.
2. Проверьте, обновляется ли `logs/wrapper.log`.

## Нормальные сообщения

### `[START] Write param successful`

Нормальный признак успешного применения параметров запуска.

### `[START] Process exit with code 0`

Нормальное завершение процесса (в том числе после ручного закрытия игры).

## Частые проблемы

### Игра стартует без эффекта от пресета

- проверьте, что `Install` выполнен успешно;
- убедитесь, что антивирус не удалил/заблокировал `service.exe`;
- выполните `Uninstall` -> `Install` повторно.

### Автоподбор выставил не тот пресет

- выполните `Apply Recommended Config` повторно;
- проверьте, что система не менялась (RAM/CPU режим);
- при необходимости вручную переключите пресет через `Select Config`.

## Когда заводить issue

Приложите:

- `logs/wrapper.log`;
- описание сценария (лаунчер/Steam/EGS, выбранный пресет, шаги воспроизведения).
